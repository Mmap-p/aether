# ÆTHER — Affiliate Program

## How It Works
1. Any ÆTHER user can become an Ambassador
2. They get a unique code and link from their profile settings
3. When someone signs up via their link and upgrades to Pro, 
   they earn 30% of that subscription monthly for 12 months
4. Partner tier (manually granted by admin) earns 40% recurring with no time limit
5. Rewardful tracks all of this automatically

## Dashboard Pages (already scaffolded at /affiliate)
Overview:  Total clicks, conversions, active referrals, 
           pending earnings, lifetime earnings
Links:     Their unique links, custom UTM campaigns, 
           QR code generator for each link
Earnings:  Transaction history, commission per conversion, 
           monthly totals chart
Payouts:   Request payout, history, payment method (via Rewardful)
Assets:    Brand kit download — logos, banners, copy templates

## Admin Controls (in /admin/revenue)
- See all affiliates and their performance
- Grant Partner tier manually
- Approve/reject payout requests
- Set commission rates per tier
- See total affiliate cost vs revenue generated
